# WealthHabit 🐍 

**WealthHabit** has been supercharged with a brand new, lightning-fast Python backend and a stunning **3D Glassmorphism** User Interface! It analyzes your spending patterns, identifies wealth-draining habits, and coaches you towards financial independence.

![Tech Stack](https://img.shields.io/badge/Tech Stack-FastAPI | Python | Vanilla JS-blue) ![Design System](https://img.shields.io/badge/Design System-3D Glassmorphism-purple)

---

## 🌟 What's New?

- **⚡ FastAPI Backend**: The entire application logic, including the AI Financial Coach, has been rewritten in Python using FastAPI for incredible speed and type safety.
- **💎 3D Glassmorphism UI**: 
  - True translucent glass panels with `backdrop-filter` blurred reflections.
  - Interactive **3D tilt card mechanics** that dynamically react to your mouse pointer.
  - Ambient glowing background elements for a premium, immersive aesthetic.
- **🤖 Built-in AI Financial Coach**: A simulated neural network endpoint (`/api/chat`) that provides personalized financial advice, saving strategies, and educational insights directly tailored to your spending profile.

## 🛠️ Tech Stack

- **Backend**: [FastAPI](https://fastapi.tiangolo.com/) (Python) & Uvicorn
- **Frontend**: Vanilla HTML5, **Vanilla CSS3 (Advanced 3D Transforms)**, Vanilla Javascript
- **Icons**: [Lucide](https://lucide.dev/)

## 🚀 Getting Started

### Prerequisites
Make sure you have Python 3.8+ installed.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/abhinavxkashyap/Finmoney.git
   cd Finmoney
   ```

2. Create a virtual environment and install the dependencies:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   ```

3. Start the FastAPI server:
   ```bash
   uvicorn main:app --port 3000 --reload
   ```

4. Open your browser and navigate to `http://127.0.0.1:3000` to experience the 3D application in action.
