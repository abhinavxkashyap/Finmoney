# WealthHabit 🚀 

**WealthHabit** is a premium, AI-powered personal finance and investment habit tracker. It goes beyond simple expense tracking by actively analyzing your spending patterns, identifying wealth-draining habits, and coaching you towards financial independence.

![Design System](https://img.shields.io/badge/Design System-Dark/Light Mode-purple) ![Tech Stack](https://img.shields.io/badge/Tech Stack-Next.js | Tailwind CSS-blue) ![Components](https://img.shields.io/badge/Components-Recharts | Lucide React-green)

---

## 🌟 Features

- **📊 Intelligent Dashboard**: Gain at-a-glance insights into your net worth, spending trends, and financial goals with beautiful pie and area charts.
- **💸 Expense Tracking**: Log, categorize, and track daily expenses to see exactly where your money goes. 
- **⚠️ Habit Analysis**: Automatically identify recurring unhealthy spending habits (e.g., excessive food delivery) and see their long-term impact on your wealth.
- **📈 Investment Simulators & Calculators**: Interactive tools to project your wealth growth using SIPs, FDs, and compound interest formulas.
- **🤖 AI Financial Coach**: A built-in AI assistant that provides personalized financial advice, saving strategies, and educational insights directly tailored to your spending profile.

## 🛠️ Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) with a custom Glassmorphism design system.
- **Icons**: [Lucide React](https://lucide.dev/)
- **Charts**: [Recharts](https://recharts.org/)
- **State Management**: React Hooks

## 🚀 Getting Started

### Prerequisites
Make sure you have Node.js and npm installed.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/abhinavxkashyap/Finmoney.git
   cd Finmoney
   ```

2. Install the dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:3000` to see the application in action.

## 💡 About The AI Coach
The AI coach is powered by a simulated backend API located at `src/app/api/chat/route.js`. It parses user input for keywords like "spending", "invest", and "habit" to deliver context-aware financial advice.
